Ініціалізація та налаштування

1. git init Ініціалізує новий локальний репозиторій Git в поточному каталозі. Тобто, перетворює вашу папку на Git-проект.

2. git config --global user.name "Ваше Ім'я" Встановлює глобальне ім'я користувача, яке буде використовуватися для ідентифікації ваших комітів.

3. git config --global user.email "[видалено електронну адресу]" Встановлює глобальну email-адресу користувача, яка також буде використовуватися для ідентифікації ваших комітів.

Робота зі змінами

4. git add . Додає всі змінені файли в індекс. Індекс – це як своєрідний буфер, який зберігає зміни перед тим, як їх зафіксувати.

5. git commit -m "Опис коміту" Фіксує зміни з індексу в репозиторій. Обов'язково додавайте зрозумілий опис, щоб потім легко орієнтуватися в історії змін.

Перегляд історії

6. git log Виводить список всіх комітів у хронологічному порядку.

Переміщення між комітами

7. git checkout Перемикається на вказаний коміт. Ви можете використовувати перші кілька символів хешу для короткого запису.

Створення гілок

8. git branch <ім'я_гілки> Створює нову гілку.

9. git checkout <ім'я_гілки> Перемикається на існуючу гілку.

Злиття гілок

10. git merge <ім'я_гілки> Зливає вказану гілку в поточну.
